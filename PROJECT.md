# Travel AI Planner

## Goal

Build an AI travel planning application to learn:

* ASP.NET Core
* PostgreSQL
* AI Application Engineering
* Tool Calling
* RAG
* Local LLM

## Tech Stack

### Frontend

* Next.js
* TypeScript
* Tailwind CSS

### Backend

* ASP.NET Core

### Database

* PostgreSQL

### AI

* OpenAI API

## MVP

Users can:

* Create a trip
* Generate an itinerary
* Save an itinerary
* View trip history

## Out of Scope

* Authentication
* Multi-Agent
* RAG
* Local Models
* Payments
